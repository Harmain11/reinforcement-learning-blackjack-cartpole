# Reinforcement Learning Blackjack and CartPole

## Overview
This notebook demonstrates reinforcement learning algorithms applied to Blackjack and CartPole environments. It includes model-based dynamic programming methods (value iteration, policy iteration) and model-free methods (SARSA, Q-Learning) implemented efficiently for both tabular and discretized state spaces.

## Features
- Dynamic Programming methods: Value Iteration and Policy Iteration.
- Model-free RL methods: SARSA and Q-Learning for Blackjack.
- Discretization of continuous CartPole states for tabular RL.
- Efficient, vectorized implementations focused on fast convergence.
- Aggregation of runs with different random seeds for performance evaluation.
- Visualizations of learning progress and convergence (commented examples).

## Tech Stack
- Python
- Jupyter Notebook / Google Colab
- Libraries: numpy, matplotlib, gymnasium, torch

## How to Use
1. Clone or download the repository.
2. Open the notebook in Jupyter Notebook or Google Colab.
3. Run the cells sequentially; some example runs and plots are commented out for faster startup.
4. Uncomment and configure parameters to perform experiments and visualize results.
